# Day-6-Application-Assignment

## Project Structure

```
Day-6-Application-Assignment/
├── data/
│   ├── raw/          ← original, unmodified data
│   └── processed/    ← cleaned/transformed data
├── do/               ← all Stata .do scripts
│   └── master.do     ← runs every other script in order
├── outputs/
│   ├── figures/      ← graphs and plots
│   ├── tables/       ← formatted tables
│   └── regressions/  ← regression output logs and tables
├── docs/             ← documentation and write-ups
└── references/       ← papers, codebooks, etc.
```

## How to Run

Open Stata and run `do/master.do` to execute all scripts in order.
